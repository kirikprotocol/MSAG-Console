Search and Replace

Added Ignore Case (shortcut: Control-E Control-I) and Regular Expressions (shortcut: Control-E Control-X) commands for toggling search bar settings from the keyboard.

The Incremental Search Bar and HyperSearch Bar commands in the Search menu no longer insert the currently selected text into the search bar. If you prefer this behavior, use Incremental Search for Word and HyperSearch for Word instead.

Clicking on a line in the HyperSearch Results window now selects all ocurrences of the search string in that line, not just the first occurrence.
