Added parametr: language.default
What for: to have capability to set default language for console
Type: string
Available values: are contained in the attribute "language" and optional attribute "country" of "locale" elements 
in file ..\smsc\webapps\scag\WEB-INF\locales.xml. For example: "ru", "en", "ru_RU", "en_US".
