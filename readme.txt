Примерная структура каталогов:
./bin - скомпилёные, готовые к употреблению бинарники, включая модули, библиотеки
      и т.д.
./bin/modules/* - бинарники конкретных модулей
./src - исходники и промежуточные файлы (объекники и т.д.)

./src/modules/* - исходники и промежуточные файлы (объекники и т.д.)
./src/modules/test/* - Пример модуля :)

./makefile - главный makefile. Исполняет makefile'ы остальных модулей.
./makefile.inc - инклудится makefile'ами модулей. В нём выставляются глобальные
                 переменные,  как то имя компилятора,  линкера,
                 дефолтные опции оных,  имена директория для объектников и
                 бинарников.

Просто make в корне сорцов соберёт весь проект (выполнит make для всех
модулей).

make deploy - скопирует все бинарники в ../bin/
На самом деле он выполнит make deploy для всех модулей проекта,
которые в свою очередь должны скопировать свои бинарники/либы
в $(DEPLOY)/имямодуля
