
-------------------------------------- 18.05.2006: ---------------------------------
1) Добавились константы: ICC_INITED, ICC_CONTINUED, ICC_COMPLETED
2) Добавилась сессионная переменная ICC_STATUS, в которой содержится статус текущей операции сессии.
3) Для RECEIPT_DELIVER_SM поля команды перестали быть доступны для записи - только для чтения.
4) Добавлен механизм ограничения на попытку создания pending операции для operation:wait для разных частей правил.
   (в зависимости от прилетевшей команды меняется список возможных pending операций, которые можно создать. 
   Это работает только на этапе выполнения правила, 
   так как на этапе инициализации невозможно однозначно по EventHandler установить команду)

* Требуется обновление xsd схемы для rules.
