	Это корень сырков системы, все подкаталоги за исключением коталога conf
	отображаются на соответствующие неймспесы C++
	например каталог core -> smsc::core

	Каталог conf содержит шаблоны конфигурационных файлов для построения, установки и тестирования системы

	построение(компиляция) системы производится в каталог указаный в переменной среды SMSC_BUILDDIR
	если такая переменная среды не установлена то SMSC_BUILDDIR=$(HOME)/build
